Family Tree

Описание

Описание проекта

Компьютерное приложение для создания и редактирования генеалогических деревьев.

Цель проекта

Разработать программное обеспечение, обеспечивающее удобный и интуитивно понятный интерфейс, который позволит пользователям легко создавать, редактировать и визуализировать свои генеалогические древа. Программный продукт обеспечивает удобное и наглядное представление генеалогической информации, позволяя пользователям оживить свою генеалогическую историю и просматривать в электронном виде свои родственные связи.

Содержание

Меню

Четыре основных пункта меню, при помощи которых пользователь может перемещаться по нужным таблицам и изменять их:

  1. Персоны – с помощью этого пункта можно вносить изменения в таблицу персон базы данных
  2. Места – с помощью этого пункта можно вносить изменения в таблицу мест базы данных
  3. События – с помощью этого пункта можно вносить изменения в таблицу мест базы данных
  4. Древо – с помощью этого пункта можно создавать генеалогическое древо на основе данных таблицы

А также пункт меню «Настройки», позволяющий:

  1. Быстро открыть/создать новую базу
  2. Открыть примеры готовых баз данных
  3. Открыть справку

Функции

Четыре основных функции со всеми таблицами:

  1. Добавление
  2. Изменение
  3. Удаление
  4. Просмотр

Дополнительные функции в «Персонах»:

  1. Фильтрация
  2. Сортировка
